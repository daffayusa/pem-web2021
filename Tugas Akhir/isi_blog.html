10 Game Mobile Terlaris di Dunia


Daffa Yusa Aditama
29 Maret 2021


1. Honor of Kings



Game terlaris nomor satu di dunia per Maret 2021 adalah Honor of Kings. Game MOBA besutan Tencent untuk China ini mendapat penghasilan $257,5 juta atau sekitar Rp3,7 triliun dari pembelian yang dilakukan pemainnya.
Game ini populer di pengguna device iOS. Honor of Kings tidak masuk ke dalam 10 besar game terlaris di Android.



2. PUBG Mobile



Game terlaris no 2 di dunia per Maret 2021 adalah PUBG Mobile. Game ini jadi game mobile besutan Tencent yang memiliki penghasilan tertinggi ke-2 di dunia. Di Maret 2021, PUBG mendapat pendapatan kotor sekitar $240,5 juta atau setara Rp3,4 triliun.

PUBG Mobile berada di peringkat yang sama pada game terlaris di iOS. Sementara di Android, PUBG Mobile berada di peringkat ke-9 di seluruh dunia.



3. Genshin Impact



Di posisi ke-3 game mobile paling laris di dunia adalah Genshin Impact. Game besutan miHoYo ini memang dirilis di multi platform. Jadi game ini bisa dimainkan di perangkat mobile, serta di perangkat lainnya seperti PS4 dan PC.

Game ini memang mendapat penerimaan yang baik dari gamers di seluruh dunia. Bahkan memecahkan rekor diunduh 17 juta kali hanya dalam empat hari perilisannya.



4. Uma Musume Pretty Derby



Baru dirilis pada akhir Februari 2021 lalu di Jepang oleh Cygames, game ini dengan cepat menjadi salah satu game mobile paling laris di dunia berdasarkan pembelian yang dilakukan pemain.

Uma Musume Pretty Derby adalah game personifikasi balap kuda yang juga memiliki anime dan serial manga. Game ini berada di peringkat yang sama di kategori pengguna iOS. Sementara di pengguna Android, game ini berada di peringkat ke-6.



5. Coin Master



Di peringkat ke-5 ada game mobile Coin Master yang dirilis Moon Active. Game ini sebenarnya adalah game lama yang dirilis sekitar tahun 2010. Game ini pada dasarnya adalah melakukan spin, mengumpulkan koin, dan mengembangkan sebuah desa.

Game ini populer di kalangan pengguna smartphone Android. Sementara di kalangan pengguna iOS, game ini tidak masuk dalam kategori 10 besar dunia.



6. Roblox



Di peringkat ke-6 ada platform game Roblox. Platform game ini memang populer di kalangan anak-anak dan remaja. Apalagi Roblox juga bisa dimainkan di multi platform. Di kalangan pengguna iOS, Roblox menempati peringkat ke-5, sementara di Android, Roblox tidak masuk ke dalam peringkat 10 besar.



7. Pokemon Go



Selanjutnya ada Pokemon GO yang masuk peringkat ke-7 game mobile terlaris di dunia per Maret 2021. Kepopuleran game ini memang sudah tidak perlu diragukan lagi, termasuk di Indonesia.

Pokemon GO berada di peringkat ke-9 game laris di iOS dan peringkat ke-7 game laris di Android.



8. Garena Free Fire



Di peringkat ke-8 game paling laris di dunia ada Garena Free Fire. Secara keseluruhan, game battleground ini memang peringkat ke-8, tapi di kalangan pengguna Android, game ini berada di peringkat ke-2 dari seluruh dunia.



9. Candy Crush Saga



Selanjutnya ada Candy Crush Saga di peringkat ke-9 game paling populer di seluruh dunia. Game ini memang dinilai bisa bikin ketagihan sejak diluncurkan pada November 2012 lalu. Meski sudah terbilang lama, game ini masih populer di kalangan gamers mobile.



10. Rise of Kingdoms



Di peringkat ke-10 ada game perang Rise of Kingdoms. Game Real Time Strategy ini membuat gamers bisa merasakan serunya membangun kerajaan, melatih pasukan, dan menjalankan misi.
